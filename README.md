# AI ATS Resume Analyzer 🤖📄

An AI-powered Resume Analyzer built using Django, MySQL, and OpenAI API.  
This project evaluates resumes against job descriptions and provides ATS score, skill matching, and improvement suggestions.

---

## 🚀 Features

- Upload Resume (PDF/DOCX)
- AI-based Resume Analysis
- ATS Score Calculation
- Skill Gap Detection
- Job Description Matching
- Suggestions to Improve Resume

---

## 🛠️ Tech Stack

- Python
- Django
- MySQL
- HTML, CSS, Bootstrap
- OpenAI API
